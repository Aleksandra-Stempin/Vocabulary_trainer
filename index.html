Game Instructions

To start learning, upload a file with extension .txt" and select the game level from dropdown list.
The file format should be "word in a foreign language";"word in your language".

The number ~ matches the number of letters in the word to be guessed.

You have the option to guess either a letter or the entire word.
To guess a letter, type the letter into the text field and click the Guess  Letter button.
To guess a word, type the word in the text field and click the Guess word button.
To find out the word to be guessed, click the Give up button.
To start a new game click the New Game button.

If you want to change the file, refresh the page.
